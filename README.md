Generate creative song lyrics using GPT and Streamlit.

## How to run

1. **Install dependencies**  
   `pip install -r requirements.txt`

2. **Set your OpenAI API key:**  
   ```
   export OPENAI_API_KEY=sk-...   # for Unix/Mac
   set OPENAI_API_KEY=sk-...      # for Windows
   ```

3. **Run Streamlit app:**  
   `streamlit run app.py`

## Features

- Customizable song theme, structure (Verse/Chorus)
- Web-based, easy to use
